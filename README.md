# AWS Cloud Portfolio

This repository contains four portfolio scenarios derived from the AWS Cloud Quest roles in Data Analytics and Solutions Architect.
Each scenario includes a description, architecture diagrams, code samples, and screenshots documenting the process.

## Scenarios
1. **Data Lake & Data Catalog**: Building a secure data lake with AWS Glue and Athena.
2. **Event-Driven ETL & Streaming Analytics**: Implementing an event-driven pipeline and streaming analytics.
3. **RESTful API with Lambda & DynamoDB**: Creating a serverless REST API with authentication.
4. **Containerized Microservices on ECS Fargate**: Deploying microservices with scalability and high availability.
