# Scenario 3: RESTful API with Lambda & DynamoDB

## Overview
This scenario focuses on designing a serverless REST API using Amazon API Gateway, AWS Lambda, and Amazon DynamoDB. Authentication is handled via Amazon Cognito.

## Steps
1. **Define API endpoints** using Amazon API Gateway.
2. **Implement business logic** in AWS Lambda functions.
3. **Persist data** in Amazon DynamoDB tables.
4. **Secure the API** using Amazon Cognito user pools for authentication.
5. **Configure monitoring & logging** using Amazon CloudWatch.
